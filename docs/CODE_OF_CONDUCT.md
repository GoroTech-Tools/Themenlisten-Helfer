# Verhaltenskodex

Wir möchten eine respektvolle und konstruktive Zusammenarbeit.

## Unsere Standards

- respektvoller Umgangston
- sachliches Feedback
- transparente Kommunikation
- keine diskriminierenden oder abwertenden Inhalte

## Durchsetzung

Bei Verstößen kann das Maintainer-Team Beiträge ablehnen oder weitere Maßnahmen treffen.

## Meldung

Bitte Vorfälle über die internen Projektkanäle bzw. über ein vertrauliches Issue melden.
