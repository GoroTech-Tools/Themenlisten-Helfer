# Release Notes v1.0.3

## Enthalten

- E-Mail-Body wird bei vorhandenen Zusatzanhängen automatisch um einen PS-Hinweis ergänzt:
  - `PS: Bitte sehen Sie sich auch die anderen Anlagen von dieser Nachricht an.`
- Keine Dopplung des Hinweistextes, falls dieser bereits im Body vorhanden ist.
- Erweiterte Testabdeckung für die BodyText-Logik bei Zusatzanhängen.
